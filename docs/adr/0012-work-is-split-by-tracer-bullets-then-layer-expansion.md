# Work Is Split By Tracer Bullets Then Layer Expansion

Implementation work will start with one or two vertical tracer bullets that cut through source ingestion, storage, retrieval, graph updates, workflow execution, UI, and evals. Once the contracts are proven by those slices, simpler agents can expand individual layers such as extractors, artifact types, graph queries, eval cases, and UI panels.

## Consequences

Higher-capability models should handle the first architecture scaffolding and tracer bullets because they must coordinate cross-layer contracts. Lower-capability agents can safely take bounded layer tasks after the tracer bullets establish interfaces, fixtures, and regression gates.
