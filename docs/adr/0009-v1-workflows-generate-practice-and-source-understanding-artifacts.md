# V1 Workflows Generate Practice And Source Understanding Artifacts

The first workflow set will cover practice and source understanding: create exercises, lessons and explanations, hints, source or chapter summaries, concept and prerequisite extraction, and quizzes. A key starter workflow selects ingested sources, similar to NotebookLM source selection, and generates the content needed by the existing PracDaGo practice UI: lesson sections, problem text, hints, starter code, solution code, tests, examples, tags, and chapter/topic metadata.

## Consequences

The workflow system must be expandable toward Studio-like artifacts such as mind maps, reports, flashcards, slide decks, and data tables, and later toward user-authored custom workflows. V1 should still ship useful starter workflows so the learner is not forced to design prompts before using the platform.
